## 4. Concept of operation
#### 1) Register User
| 구분 | 내용 |
| :--- | :--- |
| **Purpose** | 사용자가 시스템을 이용하기 위해 회원 정보를 등록한다. |
| **Approach** | 사용자는 이름, 이메일, 비밀번호 등의 정보를 입력하여 회원가입을 진행한다. 입력된 정보는 데이터베이스에 저장된다. |
| **Dynamics** | 앱을 처음 사용하는 사용자인 경우 |
| **Goals** | 사용자가 시스템의 기능을 이용할 수 있도록 계정을 생성한다. |

#### 2) Login
| 구분 | 내용 |
| :--- | :--- |
| **Purpose** | 앱을 사용하기 위해 등록된 사용자인지 확인한다. |
| **Approach** | 사용자가 앱을 실행 후 로그인 시, ID, PW를 입력 후 로그인을 요청하면 서버에서 회원 정보를 조회 후 로그인 성공/실패 여부를 확인한다. |
| **Dynamics** | 앱 실행 시 로그인할 경우 |
| **Goals** | 로그인 기능을 구현한다. |

#### 3) Upload Waste Image
| 구분 | 내용 |
| :--- | :--- |
| **Purpose** | 사용자가 쓰레기 사진을 업로드하여 종류를 분석하도록 한다. |
| **Approach** | 사용자는 카메라 또는 파일 업로드 기능을 이용하여 이미지를 등록한다. 시스템은 해당 이미지를 분석 모듈로 전달한다. |
| **Dynamics** | 사용자가 특정 폐기물의 재질이나 품목을 육안으로 구분하기 어렵거나 정확한 분리배출 규정을 즉석에서 확인하고자 하는 경우 |
| **Goals** | 쓰레기 종류를 자동으로 식별한다. |

#### 4) View Recycling Guide
| 구분 | 내용 |
| :--- | :--- |
| **Purpose** | 사용자가 올바른 분리배출 방법을 확인할 수 있도록 한다. |
| **Approach** | 시스템은 분석 결과를 기반으로 해당 쓰레기의 분리배출 방법을 화면에 표시한다. |
| **Dynamics** | AI 분석이 완료된 후 사용자가 해당 품목을 비우고 헹구고 분리하고 섞지 않는 구체적인 실천 방법을 상세히 안내받고자 하는 경우 |
| **Goals** | 사용자가 정확한 분리배출 방법을 이해하도록 돕는다. |

#### 5) Check Local Rules
| 구분 | 내용 |
| :--- | :--- |
| **Purpose** | 사용자가 지역별 분리배출 규칙을 확인할 수 있도록 한다. |
| **Approach** | 사용자가 지역을 선택하면 해당 지역의 분리배출 규칙을 데이터베이스에서 불러와 표시한다. |
| **Dynamics** | 거주지를 이동했거나 지자체마다 상이한 수거 요일, 종량제 봉투 배출 위치, 대형 폐기물 처리 방식 등을 정확히 확인해야 하는 경우 |
| **Goals** | 지역 규정을 정확히 따르도록 돕는다. |
