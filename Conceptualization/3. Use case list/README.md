### 3. Use case list

#### 1) Register User
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | User |
| **Description** | 사용자는 시스템을 사용하기 위해 회원가입을 진행한다. |

#### 2) Login
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | User |
| **Description** | 등록된 사용자는 아이디와 비밀번호를 통해 로그인한다. |

#### 3) Upload Waste Image
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | User |
| **Description** | 사용자는 쓰레기의 사진을 촬영하거나 업로드하여 시스템에 전달한다. |

#### 4) Identify Waste Item
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | System |
| **Description** | 시스템은 업로드된 이미지를 분석하여 쓰레기의 종류를 식별한다. |

#### 5) View Recycling Guide
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | User |
| **Description** | 사용자는 식별된 쓰레기의 분리배출 방법을 확인할 수 있다. |

#### 6) Check Local Recycling Rules
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | User |
| **Description** | 사용자는 자신의 지역에 맞는 분리배출 규칙을 확인할 수 있다. |

#### 7) Search Recycling Information
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | User |
| **Description** | 사용자는 특정 물품의 재활용 정보를 검색할 수 있다. |

#### 8) Manage Recycling Data
| 항목 | 내용 |
| :--- | :--- |
| **Actor** | Admin |
| **Description** | 관리자는 분리배출 방법과 관련된 데이터를 수정하거나 추가할 수 있다. |
