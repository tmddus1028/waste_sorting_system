## 6. Glossary
- 이미지 인식 (Image Recognition)
: 사용자가 업로드한 이미지를 분석하여 객체(쓰레기 종류)를 식별하는 기술

- 분리배출 (Waste Sorting)
: 쓰레기를 종류별(일반쓰레기, 재활용, 음식물 등)로 구분하여 배출하는 행위

- 머신러닝 모델 (Machine Learning Model)
: 데이터를 학습하여 새로운 입력에 대해 예측 또는 분류를 수행하는 알고리즘

- 데이터셋 (Dataset)
: 머신러닝 모델 학습 및 테스트를 위해 사용되는 이미지 및 라벨 데이터 집합

- 정확도 (Accuracy)
: 전체 예측 중 올바르게 분류된 비율로 모델 성능을 평가하는 지표

- 웹 기반 시스템 (Web-based System)
: 인터넷 브라우저를 통해 접근 및 사용이 가능한 시스템

- 사용자 인터페이스 (UI, User Interface)
: 사용자가 시스템과 상호작용하는 화면 및 구성 요소

- 데이터베이스 (Database)
: 분리배출 규칙, 사용자 정보 등을 저장하고 관리하는 시스템

- 지역별 분리배출 규칙
: 각 지자체에서 정한 쓰레기 분류 및 배출 기준

- 클라이언트 (Client)
: 서비스를 사용하는 사용자 측 환경(웹 브라우저)

- 서버 (Server)
: 클라이언트 요청을 처리하고 데이터를 제공하는 시스템
