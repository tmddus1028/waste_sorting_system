## 5. Problem statement
본 시스템은 쓰레기 분리배출을 쉽게 안내하기 위한 웹 기반 서비스이다. 
사용자는 이미지를 업로드하여 쓰레기 종류를 판별하고 이에 맞는 분리배출 방법을 안내받을 수 있도록 설계한다. 
그러나 시스템 구현 과정에서 아래와 같은 문제점이 발생할 수 있다.

### 1. 이미지 인식 정확도 문제 
이미지 기반 분류 모델은 촬영 환경(조명, 각도, 배경 등)에 영향을 받기 때문에 정확도가 저하될 수 있다. 이로 인해 잘못된 쓰레기 분류 결과가 제공될 경우 사용자가 잘못된 방식으로 분리배출을 하게 되어 환경 오염 및 시스템 신뢰도 저하로 이어질 수 있다.

### 2. 분리배출 규칙의 지역 의존성
쓰레기 분리배출 기준은 지역 및 지자체 정책에 따라 상이하며 정책이 주기적으로 변경될 수 있다. 따라서 최신 정보를 반영하지 못할 경우 사용자에게 부정확한 안내를 제공하게 되며 이는 서비스의 실효성을 떨어뜨리는 주요 요인이 된다.

### 3. 네트워크 의존성 문제
본 시스템은 웹 기반으로 동작하기 때문에 인터넷 연결이 필수적이다. 네트워크 환경이 불안정하거나 연결이 불가능한 경우 서비스 이용이 제한되며 이는 사용성 저하로 이어질 수 있다.

### 4. 사용자 입력 데이터의 신뢰성 문제
사용자가 업로드하는 이미지가 흐리거나 대상이 명확하지 않을 경우 시스템이 정확한 판단을 내리기 어렵다. 특히 여러 종류의 쓰레기가 혼합된 이미지의 경우 잘못된 결과를 초래할 가능성이 높다.

### 5. 데이터 학습 범위의 한계
모든 쓰레기 유형을 학습 데이터로 포함하는 것은 현실적으로 어려우며 특정 품목에 대해서는 분류가 불가능하거나 부정확한 결과가 도출될 수 있다. 이는 서비스의 적용 범위를 제한하는 요소로 작용한다.

이러한 문제를 최소화하기 위해 지속적인 데이터 업데이트와 시스템 개선이 필요하다.
