### User
사용자는 웹 시스템을 이용하는 일반 사용자이다. 사용자는 회원가입 및 로그인 후 쓰레기 사진을 업로드하여 재활용 여부를 확인할 수 있다. 또한 시스템을 통해 분리배출 방법을 조회하고, 재활용 정보와 지역별 분리배출 규칙을 확인할 수 있다.

### System
System은 쓰레기 분리배출 AI 가이드 웹 시스템이다. 사용자가 업로드한 쓰레기 이미지를 서버로 전달하여 분석을 요청하고 서버로부터 받은 결과를 기반으로 쓰레기의 종류를 판단하여 올바른 분리배출 방법을 사용자에게 안내한다. 또한 재활용 정보와 관련 데이터를 사용자에게 제공하는 역할을 한다.

### Server
Server는 시스템의 핵심 처리 기능을 담당하는 구성 요소이다. 사용자가 업로드한 이미지를 분석하여 쓰레기의 종류를 식별하는 이미지 인식 기능을 수행하며 재활용 데이터베이스를 통해 분리배출 방법과 재활용 정보를 제공한다. 또한 지역별 분리배출 규칙 데이터를 관리하고 시스템의 요청에 따라 필요한 정보를 전달한다.

### 주요 기능
- 회원가입(Register)
- 로그인(Login) 
- 쓰레기 사진 업로드(Upload Image) 
- 분리배출 방법 조회(Recycling Guide) 
- 지역별 규칙 조회(Local Rules) 
- 재활용 정보 검색(Recycling Information)
