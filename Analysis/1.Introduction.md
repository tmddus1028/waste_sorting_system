# 1. Introduction

## 1) Summary
오늘날 환경 보호와 자원 재활용의 중요성이 점점 커지고 있지만, 실제 생활 속에서는 올바른 쓰레기 분리배출 방법을 정확히 알지 못해 어려움을 겪는 경우가 많습니다. 특히 같은 플라스틱이나 종이류처럼 보이더라도 재질이나 오염 정도에 따라 분리배출 방법이 달라지기 때문에 일반 사용자들은 어떤 기준으로 버려야 하는지 혼란을 느끼기 쉽습니다. 또한 지역마다 분리배출 규칙이 다를 수 있어 사용자가 직접 관련 정보를 검색하고 확인하는 과정은 번거롭고 불편합니다.

이러한 문제를 해결하기 위해 사용자가 쓰레기 사진을 업로드하면 해당 품목을 자동으로 식별하고 그에 맞는 올바른 분리배출 방법을 안내하는 시스템 **‘이거뭐지’** 프로젝트를 고안하였습니다. 이 시스템을 통해 사용자는 복잡한 분리배출 기준을 일일이 찾아보지 않아도 보다 쉽고 정확하게 재활용 여부와 배출 방법을 확인할 수 있습니다. 또한 지역별 분리배출 규칙과 재활용 정보도 함께 제공함으로써 일상생활에서 올바른 분리배출을 실천할 수 있도록 돕는 것을 목표로 합니다.

## 2) Describe the Features of Project
‘이거뭐지’는 이미지 인식 기반의 쓰레기 분리배출 안내 시스템으로 사용자에게 쓰레기 종류 식별, 재활용 가능 여부 확인, 올바른 분리배출 방법 안내, 지역별 규칙 조회 기능을 제공하는 시스템입니다. 사용자는 쓰레기 사진을 촬영하거나 업로드하는 것만으로 해당 물품의 종류와 처리 방법을 빠르게 확인할 수 있으며 별도로 복잡한 검색 과정을 거치지 않아도 필요한 정보를 얻을 수 있습니다.

이 시스템의 주요 특징은 사용자가 직관적으로 활용할 수 있어 실생활에서의 편의성이 높습니다. 그리고 올바른 분리배출을 유도하여 환경 보호와 자원 재활용에 기여할 수 있고 지역별 규칙과 다양한 품목 정보를 확장하여 적용할 수 있으므로 시스템의 확장성이 높습니다.

즉 ‘이거뭐지’는 단순한 정보 제공을 넘어 사용자가 일상 속에서 보다 정확하고 편리하게 분리배출을 실천하도록 지원하는 실용적인 시스템이라고 할 수 있습니다.
