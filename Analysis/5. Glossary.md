# 5. Glossary

본 문서에서 사용되는 주요 용어의 정의는 다음과 같습니다.

| Terms | Description |
| :--- | :--- |
| **Image Recognition** | 이미지 인식은 사용자가 업로드한 이미지를 분석하여 그 안에 포함된 객체를 식별하는 기술이다. 본 시스템에서는 쓰레기 이미지에서 해당 품목을 자동으로 인식하는 데 사용된다. |
| **Waste Sorting** | 쓰레기 분리배출을 의미하며, 일반쓰레기, 재활용 쓰레기, 음식물 쓰레기 등으로 구분하여 처리하는 행위를 말한다. |
| **Machine Learning Model** | 데이터를 학습하여 새로운 입력 데이터에 대해 예측이나 분류를 수행하는 알고리즘이다. 본 시스템에서는 쓰레기 이미지를 분석하고 분류하는 데 사용된다. |
| **Dataset** | 머신러닝 모델을 학습시키기 위해 사용되는 데이터 집합이다. 다양한 쓰레기 이미지와 그에 대한 라벨 정보를 포함한다. |
| **Accuracy** | 모델이 예측한 결과 중 실제로 올바르게 분류된 비율을 의미하며, 이미지 인식 성능을 평가하는 중요한 지표이다. |
| **Web-based System** | 인터넷 브라우저를 통해 접근하고 사용할 수 있는 시스템이다. 별도의 설치 없이 웹 환경에서 실행된다. |
| **User Interface (UI)** | 사용자가 시스템과 상호작용하는 화면 및 구성 요소를 의미한다. 버튼, 입력창, 화면 레이아웃 등이 포함된다. |
| **Database** | 사용자 정보, 재활용 정보, 이미지 분석 결과 등을 저장하고 관리하는 시스템이다. |
