# 4. User Interface prototype
본 시스템은 사용자가 쉽고 직관적으로 쓰레기 분리배출 정보를 확인할 수 있도록 간단하고 명확한 사용자 인터페이스를 제공한다.   
각 화면은 사용자의 주요 기능 흐름(회원가입 → 로그인 → 이미지 업로드 → 결과 확인)을 중심으로 구성된다.


##  4.1 Main Screen
<img width="1600" height="1051" alt="image" src="https://github.com/user-attachments/assets/f3db7de7-859a-45e6-af02-1c2a0c2e098f" />
메인 화면은 사용자가 시스템에 처음 접속했을 때 표시되는 첫 화면이다. 사용자는 이 화면에서 시스템의 주요 기능을 한눈에 확인할 수 있다. 

화면에는 로그인, 회원가입, 이미지 업로드, 재활용 정보 검색, 지역별 규칙 조회 메뉴가 배치된다.

사용자는 회원가입 후 로그인하여 쓰레기 이미지 업로드 기능을 사용할 수 있다. 
또한 재활용 정보 검색이나 지역별 분리배출 규칙 조회와 같은 기본 정보 확인 기능은 메인 화면에서 바로 접근할 수 있도록 구성한다. 
이를 통해 사용자는 원하는 기능을 빠르게 선택할 수 있다.

**사용방법**  
1. 로그인 또는 회원가입 후 모든 기능을 이용할 수 있습니다.
2. 검색 및 지역별 규칙 조회는 로그인 없어도 가능합니다.

## 4.2 Register Screen
<img width="966" height="1100" alt="image" src="https://github.com/user-attachments/assets/1e3e8c88-0481-43af-80c6-187e89894929" />
회원가입 화면은 새로운 사용자가 시스템을 이용하기 위해 계정을 생성하는 화면이다. 

사용자는 이름, 이메일, 비밀번호를 입력하여 회원가입을 진행한다. 입력한 이메일은 로그인 ID로 사용되며, 비밀번호는 사용자 인증을 위해 사용된다.

사용자가 모든 정보를 입력하고 회원가입 버튼을 클릭하면 시스템은 입력값의 형식과 중복 여부를 확인한다. 
이메일 형식이 올바르지 않거나 이미 가입된 이메일인 경우 오류 메시지를 표시하고 다시 입력하도록 안내한다. 
회원가입이 정상적으로 완료되면 로그인 화면으로 이동한다.

**사용방법**  
1. 이름,이메일 비밀번호를 입력합니다
2. 회원가입 버튼을 클릭합니다.
3. 가입이 완료되면 로그인화면으로 이동합니다.

## 4.3 Login Screen
<img width="813" height="1085" alt="image" src="https://github.com/user-attachments/assets/2b11eecc-2ba0-4fe3-b8d7-b96612c36880" />
로그인 화면은 등록된 사용자가 시스템에 접속하기 위해 인증을 수행하는 화면이다. 

사용자는 이메일 또는 아이디와 비밀번호를 입력한 뒤 로그인 버튼을 클릭한다.

시스템은 입력된 정보가 데이터베이스에 저장된 회원 정보와 일치하는지 확인한다. 정보가 일치하면 메인 서비스 화면으로 이동하고, 정보가 일치하지 않으면 로그인 실패 메시지를 표시한다. 
관리자의 경우 관리자 계정으로 로그인하면 데이터 관리 화면에 접근할 수 있다.

**사용방법**  
1.  이메일(아이디)과 비밀번호를 입력합니다
2. 로그인 버튼을 클릭합니다
3. 인증이 성공하면 메인 서비스 화면으로 이동합니다.


## 4.4 Upload Waste Image Screen
<img width="1031" height="1088" alt="image" src="https://github.com/user-attachments/assets/f2f83fb5-8864-43b6-87fd-d1ffad09eb62" />
이미지 업로드 화면은 사용자가 분리배출 방법을 알고 싶은 쓰레기 사진을 등록하는 화면이다. 

사용자는 카메라를 이용해 즉시 사진을 촬영하거나, 기존에 저장된 이미지 파일을 선택하여 업로드할 수 있다.

업로드 가능한 파일 형식은 JPG, PNG 등의 이미지 파일로 제한한다. 사용자가 이미지를 선택하고 업로드 버튼을 클릭하면 시스템은 해당 이미지를 서버로 전송한다. 
이후 이미지 인식 기능을 통해 쓰레기의 종류를 자동으로 분석한다. 이미지가 흐리거나 파일 형식이 올바르지 않은 경우 재업로드를 요청한다.

**사용방법**  
1. 카메라 촬영 또는 파일 선택을 합니다.
2. 업로드 버튼을 클릭하면 분석이 시작됩니다.


## 4.5 Result Screen (Indentify + Guide)
<img width="524" height="560" alt="image" src="https://github.com/user-attachments/assets/cc2bb83a-0999-415a-9fbe-04d1fccf9454" />
결과 화면은 업로드된 이미지 분석 결과를 사용자에게 보여주는 화면이다. 

시스템은 식별된 쓰레기 품목명, 재활용 가능 여부, 기본 분리배출 방법을 화면에 표시한다.

예를 들어 플라스틱 병으로 식별된 경우, 내용물을 비우고 헹군 뒤 라벨과 뚜껑을 분리하여 배출하라는 안내를 제공한다. 
사용자는 이 화면에서 기본적인 분리배출 방법을 확인할 수 있으며, 더 자세한 정보가 필요한 경우 상세 가이드 버튼을 선택할 수 있다.


**사용방법**  
1. 시스템이 분석한 쓰레기 종류와 재활용 가능여부를 확인합니다.
2. 분리배출 방법을 확인합니다.
3. 더 자세한 정보가 필요하면 상세가이드 보기를 클릭합니다.


## 4.6 Detailed Guide Screen
<img width="599" height="817" alt="image" src="https://github.com/user-attachments/assets/310e6b99-e054-48f0-898a-535c4bd630eb" />
상세 가이드 화면은 기본 결과 화면보다 더 구체적인 분리배출 방법을 제공하는 화면이다. 

이 화면에서는 세척 방법, 재질별 분리 방법, 배출 시 주의사항 등을 자세히 안내한다.

예를 들어 플라스틱 병의 경우 내용물을 완전히 비우는 방법, 라벨 제거 여부, 뚜껑 분리 여부 등을 단계별로 설명한다. 
또한 오염이 심한 경우 재활용이 어려울 수 있다는 예외 사항도 함께 제공하여 사용자가 실제 상황에 맞게 판단할 수 있도록 돕는다.

**사용방법**  
상세한 분리배출방법과 주의사항을 확인합니다.

## 4.7 Search Recycling Information Screen
<img width="695" height="836" alt="image" src="https://github.com/user-attachments/assets/81a47e8b-999d-415d-9d04-5c81122c7eda" />
재활용 정보 검색 화면은 사용자가 이미지 업로드 없이 직접 품목명을 입력하여 재활용 정보를 찾는 화면이다. 

사용자는 검색창에 종이컵, 유리병, 캔, 비닐 등 확인하고 싶은 품목명을 입력한다.

시스템은 입력된 검색어와 관련된 재활용 정보를 검색하여 결과 목록을 표시한다. 
사용자는 검색 결과에서 원하는 품목을 선택해 재활용 가능 여부와 배출 방법을 확인할 수 있다. 검색 결과가 없는 경우에는 다른 검색어를 입력하도록 안내 메시지를 제공한다.

**사용방법**  
1. 검색어를 입력하고 검색버튼을 클릭합니다
2. 검색 결과에서 원하는 품목의 정보를 확인합니다.

## 4.8 Check Local Recycling Rules Screen
<img width="661" height="829" alt="image" src="https://github.com/user-attachments/assets/86d7b9be-517b-4301-9acb-b48b91031ef4" />
지역별 분리배출 규칙 화면은 사용자가 자신의 거주 지역에 맞는 분리배출 기준을 확인하는 화면이다. 

사용자는 지역 선택 메뉴에서 시, 군, 구 단위의 지역을 선택한다.

시스템은 선택된 지역의 수거 요일, 배출 시간, 배출 장소, 품목별 배출 기준 등을 표시한다. 지역마다 분리배출 기준이 다를 수 있으므로, 이 화면은 사용자가 자신의 지역 규칙에 맞게 쓰레기를 배출하도록 돕는 역할을 한다.

**사용방법**  
1. 지역을 선택하면 해당 지역의 규칙이 표시됩니다.
2. 수거 요일, 배출시간, 장소를 확인합니다.

## 4.9 Admin Data Management Screen
<img width="760" height="827" alt="image" src="https://github.com/user-attachments/assets/9d2afe20-218c-4902-97ac-28e39d51f500" />
관리자 데이터 관리 화면은 관리자가 시스템의 재활용 정보와 지역별 규칙 데이터를 관리하는 화면이다. 

관리자는 품목별 분리배출 정보, 상세 가이드, 지역별 수거 규칙 등을 추가, 수정, 삭제할 수 있다.

예를 들어 새로운 재활용 품목이 추가되거나 지자체의 배출 기준이 변경된 경우, 관리자는 해당 정보를 수정하여 시스템에 반영한다.
이를 통해 사용자는 항상 최신 분리배출 정보를 제공받을 수 있다.

**사용방법**  
1. 품목 데이터를 추가,수정,삭제 할 수 있습니다
2. 변경한 내용은 데이터베이스에 저장됩니다.
